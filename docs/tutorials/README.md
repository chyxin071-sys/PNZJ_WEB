## 🔥 核心概念速查

### 微信小程序是什么？
- 一种不需要下载安装的应用
- 在微信里直接打开使用
- 体验接近原生 App
- 开发成本比 App 低很多

### 适合做什么？
✅ **适合**：
- 工具类应用（计算器、查询工具）
- 电商类应用（商城、外卖）
- 内容类应用（新闻、资讯）
- 企业内部管理系统
- O2O 服务（预约、上门服务）

❌ **不适合**：
- 游戏（有专门的小游戏）
- 社交应用（微信本身就是社交）
- 需要复杂计算的应用
- 需要后台常驻的应用

### 云开发 vs 自建后端
| 对比项 | 云开发 | 自建后端 |
|--------|--------|---------|
| 开发难度 | 简单 | 复杂 |
| 运维成本 | 低（自动运维） | 高（需要自己维护） |
| 费用 | 按量计费 | 服务器固定费用 |
| 扩展性 | 自动扩容 | 需要手动扩容 |
| 适用场景 | 中小型项目 | 大型复杂项目 |

### 开发流程
```
1. 注册小程序账号
   ↓
2. 完成认证（个人/企业）
   ↓
3. 获取 AppID
   ↓
4. 安装开发者工具
   ↓
5. 创建项目
   ↓
6. 开发功能
   ↓
7. 测试
   ↓
8. 提交审核
   ↓
9. 发布上线
```

## 📖 推荐阅读顺序

**第零天（准备阶段）**：
- 阅读 [从需求到PRD](./00_从需求到PRD.md)
- 用 AI 梳理你的项目需求
- 编写简单的 PRD
- 设计数据库结构

**第一天**：
- 阅读 [微信小程序完全指南](./01_MINIPROGRAM_COMPLETE_GUIDE.md) 前半部分
- **重点看"开发前必读：6大铁律"**
- 注册小程序账号
- 安装开发者工具
- 创建第一个 Hello World 项目

**第二天**：
- 阅读 [微信小程序完全指南](./01_MINIPROGRAM_COMPLETE_GUIDE.md) 后半部分
- 学习基本语法（WXML、WXSS、JS）
- 做一个简单的页面

**第三天**：
- 阅读 [云开发入门指南](./02_CLOUD_DEVELOPMENT_GUIDE.md)
- 开通云开发
- 学习云数据库基本操作

**第四天**：
- 继续学习云开发（云存储、云函数）
- 做一个带数据库的小项目

**第五天**：
- 阅读 [常见坑与解决方案](./03_COMMON_PITFALLS.md)
- 了解常见问题
- 提前避坑

**第六天及以后**：
- 根据自己的项目需求
- 参考 [最佳实践指南](./04_BEST_PRACTICES.md)
- 或 [从0到1实战案例](./06_REAL_WORLD_CASE.md)
- 如果用 AI 开发，看 [AI 辅助开发指南](./05_AI_ASSISTED_DEVELOPMENT.md)
- 开始开发自己的项目
