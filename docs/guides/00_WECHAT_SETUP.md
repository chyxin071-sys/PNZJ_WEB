## 1. 账号注册与认证

### 1.1 个人认证 vs 企业认证对比

| 对比项 | 个人认证 | 企业认证 |
|--------|---------|---------|
| **认证费用** | 免费 | 300元/年 |
| **认证周期** | 即时 | 3-7个工作日 |
| **所需材料** | 身份证 | 营业执照、对公账户、法人信息 |
| **功能限制** | ❌ 无支付功能<br>❌ 无卡券功能<br>❌ 部分接口受限 | ✅ 完整功能 |
| **用户信任度** | 较低（显示个人主体） | 较高（显示企业主体） |
| **适用场景** | 个人项目、学习测试 | 商业项目、企业应用 |

**本项目选择**：企业认证
**原因**：
- 需要获取用户手机号（个人认证无此权限）
- 需要发送订阅消息
- 商业项目需要企业主体背书

### 1.2 企业认证完整流程

#### 步骤 1：注册小程序账号
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击"立即注册" → 选择"小程序"
3. 填写邮箱（注意：一个邮箱只能注册一个小程序）
4. 激活邮箱并设置密码
5. 选择主体类型：**企业**

#### 步骤 2：填写企业信息
需要准备的材料：
- 营业执照注册号
- 企业对公账户
- 管理员身份证信息
- 管理员手机号（需接收验证码）

#### 步骤 3：企业认证
1. 登录小程序后台 → 设置 → 微信认证
2. 支付 300 元认证费（微信支付）
3. 填写认证资料：
   - 上传营业执照扫描件
   - 填写企业基本信息
   - 填写运营者信息
   - 填写发票信息（可选）

4. 等待审核（通常 3-5 个工作日）
5. 审核通过后会收到通知

#### 步骤 4：完善小程序信息
1. 设置 → 基本设置 → 填写小程序名称、简介、头像
2. 类目管理 → 添加服务类目（根据业务选择）
3. 获取 AppID（开发 → 开发管理 → 开发设置）

### 1.3 认证过程中的常见问题

**Q1：对公账户打款验证失败？**
- 确认账户名称与营业执照完全一致
- 打款金额通常在 1 元以内，注意查收
- 打款备注中有验证码，需要在后台填写

**Q2：法人扫脸验证失败？**
- 确保光线充足
- 按提示做动作（眨眼、转头等）
- 如果多次失败，可以选择"对公账户打款验证"

**Q3：营业执照被占用？**
- 一个营业执照可以注册 50 个小程序
- 如果提示被占用，检查是否之前注册过
- 可以在"账号复用"中关联已有账号

**Q4：认证审核被驳回？**
- 常见原因：资料不清晰、信息不一致、类目选择错误
- 根据驳回原因修改后重新提交
- 可以联系客服（400-882-9911）咨询

## 2. 小程序基础配置

### 2.1 开发者权限管理

**路径**：管理 → 成员管理

**角色说明**：
- **管理员**：最高权限，可以管理所有设置
- **开发者**：可以开发、调试、上传代码
- **体验者**：可以体验未发布的版本
- **数据分析者**：只能查看数据统计

**操作步骤**：
1. 添加开发者：输入微信号 → 选择权限 → 发送邀请
2. 被邀请人在微信中接受邀请
3. 开发者可以在开发者工具中登录

### 2.2 服务器域名配置

**路径**：开发 → 开发管理 → 开发设置 → 服务器域名

**需要配置的域名类型**：
- **request 合法域名**：API 请求域名
- **socket 合法域名**：WebSocket 域名（如果用到）
- **uploadFile 合法域名**：文件上传域名
- **downloadFile 合法域名**：文件下载域名

**本项目配置**：
```
request 合法域名：
- https://你的Web端域名（如果有）
- https://tcb-api.tencentcloudapi.com（云开发API）

uploadFile 合法域名：
- https://你的云存储域名
```

**注意事项**：
- 必须是 HTTPS 协议
- 不能使用 IP 地址
- 不能带端口号
- 域名需要备案（国内服务器）
- 开发阶段可以在开发者工具中勾选"不校验合法域名"

### 2.3 订阅消息模板申请

**路径**：功能 → 订阅消息

**本项目使用的模板**：
1. **工地进度更新通知**（PROJECT_UPDATE）
   - 用途：工序验收、开工通知
   - 字段：工地名称、进度状态、更新时间

2. **待办提醒**（TODO_REMINDER）
   - 用途：新待办分配、待办到期提醒
   - 字段：待办内容、截止时间、执行人

3. **客户状态变更**（LEAD_STATUS_CHANGE）
   - 用途：客户状态变化通知
   - 字段：客户姓名、状态变更、变更时间

4. **设计节点完成**（DESIGN_NODE_COMPLETE）
   - 用途：设计工作流节点完成
   - 字段：节点名称、完成时间、负责人

5. **查看权限申请**（SHARE_ACCESS_REQUEST）
   - 用途：业主申请查看工地进度
   - 字段：申请人、工地名称、申请时间

**申请步骤**：
1. 在模板库中搜索关键词（如"进度更新"）
2. 选择合适的模板 → 添加
3. 记录模板 ID（代码中需要用到）
4. 在 `utils/subscribe.js` 中配置模板 ID

### 3.2 开通云开发流程

#### 步骤 1：在小程序后台开通
1. 登录小程序后台
2. 开发 → 云开发 → 开通
3. 选择套餐（可以先选免费版，后续升级）
4. 创建环境（环境名称自定义，如 `prod-xxx`）
5. 记录环境 ID（代码中需要用到）

#### 步骤 2：在开发者工具中初始化
1. 打开微信开发者工具
2. 点击"云开发"按钮
3. 进入云开发控制台
4. 确认环境已创建成功

### 3.3 云数据库配置

#### 创建集合（表）
本项目需要的集合：

```javascript
// 核心业务集合
- users          // 员工账号
- leads          // 客户线索
- projects       // 工地项目
- quotes         // 报价单
- contracts      // 合同（暂未使用）
- materials      // 材料大厅
- inventory      // 库存记录
- followUps      // 跟进记录
- todos          // 待办事项
- notifications  // 通知消息
- shareAccess    // 分享查看权限申请
```

**创建步骤**：
1. 云开发控制台 → 数据库
2. 点击"添加集合"
3. 输入集合名称
4. 重复以上步骤创建所有集合

#### 权限配置

**重要**：默认权限是"仅创建者可读写"，需要修改！

**本项目权限设置**：
```javascript
// 大部分集合：所有用户可读写（因为小程序端直接操作数据库）
users, leads, projects, quotes, materials, inventory,
followUps, todos, notifications, shareAccess
→ 权限设置为"所有用户可读写"

// 注意：这样设置安全性较低，生产环境建议用云函数+权限校验
```

**如何修改权限**：
1. 数据库 → 选择集合 → 权限设置
2. 选择"所有用户可读写"
3. 保存

### 3.4 云存储配置

**用途**：存储图片、文件（报价单附件、工地照片等）

**配置步骤**：
1. 云开发控制台 → 存储
2. 默认已开通，无需额外配置
3. 记录存储桶名称（上传文件时需要）

**权限设置**：
- 默认"仅创建者可读写"
- 如果需要公开访问（如分享页图片），需要设置为"所有用户可读"

### 3.5 云函数创建

**本项目使用的云函数**：

1. **sendSubscribeMessage**
   - 功能：发送订阅消息
   - 原因：订阅消息只能在云函数中调用

2. **getPhoneNumber**
   - 功能：获取用户手机号
   - 原因：获取手机号需要在服务端解密

3. **login**
   - 功能：获取用户 openid
   - 原因：openid 只能在服务端获取

**创建步骤**：
1. 在项目中创建 `cloudfunctions` 文件夹
2. 在文件夹中创建云函数目录（如 `sendSubscribeMessage`）
3. 在云函数目录中创建 `index.js` 和 `package.json`
4. 在开发者工具中右键云函数目录 → 上传并部署

**部署注意事项**：
- 云函数修改后必须重新上传部署才能生效
- 可以选择"云端安装依赖"（自动安装 npm 包）
- 部署后在云开发控制台可以看到函数列表

## 4. 开发工具准备

### 4.1 微信开发者工具

**下载地址**：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

**版本选择**：稳定版（Stable Build）

**安装步骤**：
1. 下载对应系统的安装包
2. 安装（Windows 直接运行 exe，Mac 拖到 Applications）
3. 打开工具 → 微信扫码登录
4. 导入项目 → 选择项目目录 → 输入 AppID

**常用功能**：
- **编译**：Ctrl/Cmd + B
- **预览**：生成二维码，手机扫码体验
- **真机调试**：可以在手机上调试并查看日志
- **云开发控制台**：管理数据库、存储、云函数
- **调试器**：查看 Console、Network、Storage

### 4.2 VS Code 插件推荐

**必装插件**：
- **微信小程序开发工具**（minapp）：语法高亮、代码提示
- **WXML - Language Service**：wxml 文件支持
- **小程序助手**：代码片段、快速创建页面

**可选插件**：
- **Prettier**：代码格式化
- **ESLint**：代码检查
- **Path Intellisense**：路径自动补全

### 4.3 调试技巧

**小程序端调试**：
1. 开发者工具 → 调试器 → Console
2. 使用 `console.log()` 打印日志
3. 真机调试可以看到真实环境的日志

**云函数调试**：
1. 云开发控制台 → 云函数 → 选择函数 → 测试
2. 输入测试参数 → 运行
3. 查看返回结果和日志

**数据库调试**：
1. 云开发控制台 → 数据库 → 选择集合
2. 可以直接查看、编辑、删除数据
3. 可以执行查询语句测试
